# Flutter Weather App

**A responsive weather application built with Flutter as part of an academic project.**  
It displays weather information for multiple cities with dynamic backgrounds and adaptive layouts for both mobile and web.

## Features

- **Dynamic backgrounds** based on weather conditions  
- **Responsive design** for different devices and orientations  
- **Preloaded weather data** for Algiers, Skikda, Oran, and Constantine  
- **Swipe or keyboard navigation** between cities  
- **Horizontally scrollable 5-day forecast**  
- **Stable layout handling** text overflow and resizing  

## Technologies

- **Flutter**  
- **Dart**
